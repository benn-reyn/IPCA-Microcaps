# IPCA in MicroCaps

## Requirements

- Python 3.x
- Jupyter Notebook

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/benn-reyn/IPCA-Microcaps.git
   cd IPCA-Microcaps
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Open and run `IPCAfinal` in Jupyter Notebook to perform the analysis.
